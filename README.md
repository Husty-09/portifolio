## 🚀 Rodar Localmente

### Requisitos
- Node.js 18+
- npm ou yarn

### Passos

```bash
# Clonar repositório
git clone https://github.com/Husty-09/portfolio.git
cd portfolio

# Instalar dependências
npm install

# Rodar servidor de desenvolvimento
npm run dev
```

Acessa [http://localhost:3000](http://localhost:3000)

### Build para produção

```bash
npm run build
npm start
```
